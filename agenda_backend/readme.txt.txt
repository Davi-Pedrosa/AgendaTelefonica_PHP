- instalar dependências:
pip install -r requisitos.txt

- instalar supabase
pip install supabase

- executar backend:
python app.py

- acesso navegador:
http://127.0.0.1:5000


endpoints:
GET /contatos → lista contatos

POST /contatos → adiciona novo contato

PUT /contatos/<id> → atualiza contato

DELETE /contatos/<id> → remove contato

GET /categorias / POST /categorias
